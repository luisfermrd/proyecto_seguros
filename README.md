## Install Dependencies

```
composer install
```

```
npm install
```

## Generating Migrations

```
php artisan make:migration create_name_table
```

## Running Migrations

```
php artisan migrate
```

## Drop All Tables & Migrate
```
php artisan migrate:fresh
```

```
php artisan migrate:fresh --seed
```

## Ejecutar proyecto

```
php artisan serve
```

## Start the development server

```
npm run dev
```
